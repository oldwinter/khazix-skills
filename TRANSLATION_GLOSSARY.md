# 中文化术语表

这个仓库以中文说明为主，但保留会影响识别、安装、运行或品牌归属的专有名词。

| 原词 | 中文写法 / 处理方式 |
|---|---|
| Skill | 保留 `Skill`，指 Agent 可加载的结构化指令集 |
| Agent | 保留 `Agent`，必要时解释为 AI agent |
| Agent Skills | 保留 `Agent Skills`，指开放标准 |
| Claude Code / Codex / OpenCode / OpenClaw | 品牌与 runtime 名称，保留原文 |
| AI HOT / aihot.virxact.com | 产品名与域名，保留原文 |
| neat-freak / hv-analysis / khazix-writer / aihot / storage-analyzer | skill 目录名与安装名，保留原文 |
| README.md / CLAUDE.md / AGENTS.md / docs/ | 文件名与路径，保留原文 |
| API / REST API / MCP server / User-Agent | 协议、接口和配置术语，保留原文 |
| JSON key / 函数名 / 变量名 / 命令 | 保留原文，不翻译 |
| Green / Yellow / Red tier | 中文写作「绿灯 / 黄灯 / 红灯」，必要时保留英文 tier |
| Trash / Finder / Recycle Bin | 按平台写「废纸篓 / 访达 / 回收站」，命令或 API 字段保留原文 |

翻译原则：面向用户和 agent 的自然语言改成中文；命令、路径、URL、包名、API 名、配置键、代码逻辑、license 法律表达不翻译。
